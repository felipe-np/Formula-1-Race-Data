# Dashboard de Fórmula 1 - Dataset 1950-2024

Este projeto é um dashboard de Business Intelligence referente a um dataset de Fórmula 1, que engloba o período de 1950 até 2024 com informações referentes a pilotos, corridas, circuitos, etc.

O objetivo do dashboard, assim como quem gosta do esporte, foi explorar insights e aplicar um storytelling de dados tendo como base a história da Formula 1.

Link dataset: https://www.kaggle.com/datasets/rohanrao/formula-1-world-championship-1950-2020
